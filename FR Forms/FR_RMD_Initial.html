Formulaire de Renseignement de Matière Dangereuse

Ce formulaire vous permet de transmettre les informations d'identification d'une matière
dangereuse sur une opération d'urgence.

Il inclu:

  • Le renseignement du code danger.
  • Le renseignement du code ONU.
  • Le code du pictogramme losange situé sur la pointe inférieure du panneau.
  • L'état de la matière. Il faudra péciser si la matière n'est pas présente dans la liste.
  • La couleur.
  • L'odeur. Cette information est transmise par le personnel formé. Ne vous rendez pas
    auprès du sinistre.
  • Les éventuels informations complémentaires


Vous pouvez enregistrer votre formulaire rempli dans un fichier. Il enregistrera les
informations sous forme de fichier texte avec la date et l'heure comme nom de fichier texte.
Cela vous permettra de recharger le formulaire avec TOUTES les données enregistrées.
Vous faites cela avant de transmettre le formulaire.
Ensuite, vous pouvez ouvrir un nouveau formulaire et recharger les données HTML
enregistrées et continuer. La date et l'heure seront celles enregistrées précédamment,
donc modifiez-les si nécessaire. Vous pouvez maintenant utiliser n’importe quel navigateur,
enregistrer votre formulaire, (autant de versions que vous le souhaitez),
puis remplissez à nouveau.

L'impression de cette page de visualisation de formulaire dépend de votre sélection
d'imprimante et des paramètres. Divers navigateurs Web et leur affichage les
tailles ne sont pas toutes les mêmes.
L'impression à partir d'une fonction d'impression du navigateur peut être différent
du mode d'impression par défaut du système d'exploitation.
Sélectionnez ce qui fonctionne pour vous, pour les deux :impression et visualisation.
Vous pouvez imprimer directement sur l'imprimante au format HTML ou enregistrer
en tant que PDF puis imprimez le PDF.
Dans certains cas le mode paysage peut imprimer mieux que Portrait.
